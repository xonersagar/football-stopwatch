Project Overview

This web-based stopwatch application is designed with a football theme, demonstrating my proficiency in HTML, CSS, and JavaScript. The stopwatch includes features such as start, stop, and reset functionalities, and it displays time in hours, minutes, seconds, and deciseconds. The football icon revolves when the stopwatch is running, and the background changes dynamically to add an engaging visual element. This project reflects my ability to combine functionality with a creative user interface.
